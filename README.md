# PieTurbo

## インストール方法
1. cloneする
```bash
git clone https://github.com/ryudedev/preTurbo.git
```

2. 依存関係をインストールする。
```bash
pnpm i
```

3. lintまたはformatを整える
```bash
# lint
pnpm lint

# format
pnpm format
```

4. サーバーを起動させる
```bash
pnpm build
pnpm dev
```

5. ブラウザーを確認する。
[docs](http://localhost:3000)
[web](http://localhost:3001)
[api](http://localhost:8080)
